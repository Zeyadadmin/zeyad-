# **App Name**: Yemen Ride

## Core Features:

- User Authentication: Secure login and signup using email, password, and Google authentication.
- Ride Booking: Allow users to book rides by selecting pickup and destination locations, integrated with a map interface.
- Service Listings: Display lists of available services such as travel agencies, shops, hotels, etc., with filtering and sorting options.
- Real-time Chat: Enable real-time chat between users and service providers with support for text, images, videos, and voice messages.
- User Dashboards: Provide users, drivers, and merchants with dashboards to manage bookings, orders, and analytics.
- AI-Powered Smart Assistant: Integrate a smart assistant tool to provide personalized recommendations, manage bookings, and answer user queries.

## Style Guidelines:

- Primary color: Golden/Yellow (#FFC107) (HSL 38, 92%, 50%) for main buttons, links, and attention-grabbing elements.
- Background color: Very light gray (#FAFAFA) (HSL 0, 0%, 98%) to provide a sense of spaciousness and clarity.
- Secondary color: Light gray (#F5F5F5) (HSL 0, 0%, 96.1%) for section backgrounds and cards to distinguish them from the main background.
- Body and headline font: 'Cairo' (sans-serif) for all texts in the application, providing clear and comfortable reading in Arabic.
- Use clear and simple icons from the 'Lucide React' library.
- Apply a clean, modern, and flat design style.
- Use light shadows and rounded corners for cards and buttons to give a sense of depth and softness.
